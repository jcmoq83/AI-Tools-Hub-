# AI-WEB 미니 게임 천국

## 개요

이 애플리케이션은 사용자가 여러 가지 간단한 미니 게임을 즐길 수 있는 웹사이트입니다. 처음에는 "로또 번호 생성기"와 "가위바위보 게임" 두 가지 기능을 제공하며, 현대적이고 직관적인 디자인을 통해 누구나 쉽게 사용할 수 있도록 제작되었습니다.

## 스타일, 디자인 및 기능

### 공통 요소

*   **탐색 메뉴:** 화면 상단에 고정된 메뉴를 통해 사용자가 원하는 게임을 쉽게 선택하고 전환할 수 있습니다.
*   **디자인:** 전체적으로 어두운 테마를 기반으로 하며, 각 게임의 특성에 맞는 생생한 색상과 아이콘을 사용하여 시각적 즐거움을 더합니다.
*   **레이아웃:** 콘텐츠를 중앙에 배치하고 충분한 여백을 활용하여 깔끔하고 정돈된 느낌을 줍니다.
*   **반응형:** 모바일, 태블릿, 데스크톱 등 모든 기기에서 최적화된 화면을 제공합니다.
*   **애니메이션:** 부드러운 전환 효과와 인터랙션 애니메이션을 통해 사용자 경험을 향상시킵니다.

### 1. 로또 번호 생성기

*   **기능:** "번호 생성" 버튼을 클릭하면 1부터 45까지의 숫자 중 겹치지 않는 6개의 행운의 번호를 무작위로 생성합니다.
*   **디자인:** 생성된 번호는 각각 다른 색상의 동그란 로또 볼 모양으로 표시되며, 나타날 때 부드러운 애니메이션 효과가 적용됩니다.

### 2. 가위바위보 게임

*   **기능:**
    *   사용자는 '가위', '바위', '보' 아이콘 버튼 중 하나를 선택하여 게임에 참여합니다.
    *   컴퓨터도 무작위로 하나를 선택하며, 사용자의 선택과 함께 화면에 표시됩니다.
    *   승패 결과를 "WIN", "LOSE", "DRAW" 메시지로 명확하게 알려줍니다.
    *   플레이어와 컴퓨터의 점수를 기록하고 화면에 표시하여 게임의 진행 상황을 쉽게 파악할 수 있게 합니다.
*   **디자인:** 직관적인 아이콘과 명확한 결과 표시로 게임의 몰입도를 높입니다.

## 현재 계획

1.  **HTML (`index.html`):**
    *   상단에 게임 간 전환을 위한 탐색 메뉴 구조를 추가합니다.
    *   '로또 번호 생성기'와 '가위바위보 게임'의 콘텐츠를 담을 두 개의 개별 섹션을 생성합니다.

2.  **CSS (`style.css`):**
    *   탐색 메뉴에 대한 스타일(레이아웃, 색상, 호버 효과)을 적용합니다.
    *   가위바위보 게임 섹션(버튼, 결과 표시, 점수판)에 대한 스타일을 추가합니다.
    *   게임 전환 시 한 페이지만 보이도록 하는 스타일을 정의합니다.

3.  **JavaScript (`main.js`):**
    *   탐색 메뉴 링크에 클릭 이벤트 리스너를 추가하여, 클릭된 게임에 해당하는 섹션만 화면에 표시되도록 로직을 구현합니다.
    *   가위바위보 게임 로직(사용자 선택, 컴퓨터 선택, 승패 판정, 점수 업데이트)을 구현합니다.
